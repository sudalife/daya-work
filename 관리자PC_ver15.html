⚙️ Firebase 설정
Firebase Realtime Database URL을 입력하면 모바일 앱과 실시간 연동됩니다.
설정은 이 브라우저에 저장됩니다.
🔧 Firebase 설정 방법
1. console.firebase.google.com 접속 → 새 프로젝트 생성
2. 빌드 → Realtime Database → 데이터베이스 만들기 클릭
3. 지역: asia-southeast1 선택 → 테스트 모드로 시작
4. 생성된 URL 복사 (예: https://xxx-default-rtdb.asia-southeast1.firebasedatabase.app)
5. 아래에 붙여넣기 후 저장
Firebase Realtime Database URL
